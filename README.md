## 为什么搞个这个项目？

主旨：通过看一篇文章的时间了解中共1周 1月 甚至1年来对老百姓的迫害

方便定位，统计；CCP永远篡改 ”记忆“，而且CCP的谎言最怕对比

例如，2022年4月16日网上视频 有 6-7起 被清零政策逼到自杀的，但是由于事件太久不好找到视频在哪里了

**也希望各位网友共同加入进来，共同记录中共恶行**

## 放一个文件里不乱吗？
